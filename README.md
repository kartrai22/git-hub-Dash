# 🚀 JIRA-like GitHub Board

## Setup Steps

1. Create a Project (Board view)
2. Add fields:
   - Status
   - Priority
   - Story Points
   - Sprint (Iteration)

## Status Flow

### Story/Task
Backlog → In Progress → Code Review → QA → Done

### Bug
Bug Triage → Fixing → Retest → Closed

### Subtask
In Progress → Blocked → Done

## Labels
Bug, Story, Task, Subtask, Epic
